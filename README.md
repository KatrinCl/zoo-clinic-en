🐾 VetClinic Website

A full-stack web application for a veterinary clinic with a public website and an admin panel for managing content.

🔗 Demo

Live Demo

🚀 Features

User Interface:

Browse clinic services
Price list
Doctor profiles
Photo gallery
Fully responsive (mobile, tablet, desktop)

Admin Panel:

Add/remove doctors and services
Upload images via ImageKit
Simple content management

🛠 Tech Stack:

Frontend: React (JSX), Tailwind CSS, Axios
Backend: Node.js, Express, MongoDB (Mongoose)
Additional: ImageKit (image storage), Multer (file uploads)

🎯 Purpose:

Created for learning and portfolio purposes to practice full-stack development, REST APIs, and admin panel implementation.
